zts-svctoken
============

ZTS Service JWT SVID Client application in Go to generate service jwt tokens

```shell
zts-svctoken -zts <zts-server-url> -domain <domain-name> -service <service-name> -svc-key-file <private-key-file> -svc-cert-file <service-cert-file> [-cacert <ca-cert-file>] -audience <audience> -provider <provider-name> -instance <instance-id> -nonce <nonce> [-spiffe-svid] [-spiffe-uri <uri>]
```
